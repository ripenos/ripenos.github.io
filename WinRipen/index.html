    Shortcut Keys

    1. Alt + s - Open Start Menu
    2. Alt + s < Alt + a - Open Start Menu and select App search
    3. Alt + t - Open Multitasking view
    4. Alt + w - Open Widgets panel
    5. Alt + q - Open Web search
    6. Alt + l - Set Lock
    7. Alt + r - Open Run dialog
    8. Alt + i - Open Settings
    9. Ctrl + Shift + s - Screenshot

    Run Commands

    1. winver - Open About Windows Ripen
    2. calc - Open Calculator
    3. Lock - Set Lock
    4. notes - Open Notes
    5. cmd - Opens Terminal
